# Mentor Portal – Admin Panel

This is the **admin panel** for the Hello Mentor portal. The platform functions on a role-based access system. Super admins have full control, including the ability to create coupons, while employees can upload and manage essential data for the Hello Mentor application.

The portal supports uploading and managing:

- Community application links  
- Coupons  
- Employee data  
- Closing ranks  
- Seat matrix  
- Other essential datasets  

Admins can **create**, **edit**, and **delete** all relevant data entries through the UI.

---

## Tech Stack

- **Next.js**
- **React.js**
- **Redux Toolkit Query (RTK Query)**
- **React Bootstrap**
- **React Modal**
- **SweetAlert2**
- **React Input 2**
